sevlet (Server Applet의 줄임말)
: 자바 코드들 사이에 HTML을 끼워넣는 구성방식의 웹페이지 작성
- 유지보수, 수정이 어렵다.

CGI 단점을 보완하기 위해 SUN에서 개발한 자바 기반의 웹 프로그래밍 솔루션이다.
서블릿을 기반으로 JSP가 개발됐다. JSP도 사실 내부적으로는 서블릿으로 변환되어 수행된다.
따라서 서블릿의 구성이나 동작방식을 이해하는 것이 JSP를 이해하는 데 많은 도움이 될 것이다.


jsp HTML 사이에 자바 코드들을 끼워넣는 방식으로 웹페이지 작성

Spring

Spring Boot


GET과 POST 전송방식

GET 방식
- 전송하는 데이터를 주소창에 끼워 넣어 전송
- 보안이 취약하다
- 전송 용량 제한이 작음
- 상대적으로 속도가 빠름

POST 방식
- 바디에 끼워넣어 전송
- 보안 강화 -> 안 보임
- 용량 제한 없음
- 상대적으로 속도 느림
